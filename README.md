## Description
This project, **NetPractice**, is an introduction to computer networking. Its primary goal is to teach the fundamentals of configuring small-scale networks. By completing 10 different networking levels, it provides hands-on experience in understanding how IP addressing works and how to set up networks using proper subnetting, routing, and device configurations.

## Instructions
**Running the training interface:**
To start the NetPractice training interface, run the provided shell script from your terminal:
```bash
./run.sh
```

**Exporting configurations:**
Once you have successfully completed a level, you can export your configuration directly from the training interface. These exported configurations generate the files needed for submission.

**Submission requirements:**
10 exported configuration files (one per level) must be placed at the repository root.
